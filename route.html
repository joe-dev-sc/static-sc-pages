About Routing in Static Sites

In static sites like GitHub Pages, "routing" is simply navigating between different HTML files. Each HTML file acts as a separate page or route.

For more complex routing needs, you might need to use JavaScript frameworks or GitHub Pages with Jekyll.

You can add more routes by creating additional HTML files and linking to them in the navigation.
